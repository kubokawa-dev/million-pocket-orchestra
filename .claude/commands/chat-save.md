# /chat-save

このチャットのやり取り内容を `.chat-history/` ディレクトリに詳細に永続化・保存してください。

## 引数（任意）
- `title=<text>`: ファイル名に使うタイトルを指定（省略時は会話内容から自動生成）

## ゴール
- 現在のチャットセッション全体の内容を、Markdown形式で永続化する
- ファイル名: `YYYYMMDD-{チャットの内容から最適化されたタイトル}.md`
- 保存先: `.chat-history/` ディレクトリ

## 進め方（必須手順）

### 1. 保存先ディレクトリの確認・作成
```bash
mkdir -p .chat-history
```

### 2. ファイル名の生成
- 日付部分: 現在の日付を `YYYYMMDD` 形式で取得
- タイトル部分:
  - `title=` が指定されていればそれを使用
  - 指定がなければ、会話内容を分析して以下を考慮してタイトルを生成:
    - 主な作業内容・目的
    - 使用した技術・ツール
    - 解決した問題
  - タイトルは日本語OK、ファイル名として使える文字のみ（スペースはハイフンに置換）
  - 20〜50文字程度が目安

### 3. Markdownファイルの内容構成

以下の構成で詳細に記録する:

```markdown
# {タイトル}

- **日時**: YYYY-MM-DD HH:MM
- **セッションID**: （あれば）
- **主なトピック**: タグ形式で列挙

---

## 📋 サマリー

このチャットセッションで行った作業の要約を3〜5行で記載。

## 🎯 目的・背景

ユーザーが解決したかった課題や目的を記載。

## 💬 やり取りの詳細

### ユーザーからの質問・依頼 1
> ユーザーの発言を引用形式で記載

**アシスタントの対応:**
- 行った分析
- 提案した解決策
- 実行したコマンドやコード変更

### ユーザーからの質問・依頼 2
> ...

（以降、主要なやり取りを順に記載）

## 🔧 実行した主な操作

### ファイル操作
- 作成したファイル: `path/to/file`
- 編集したファイル: `path/to/file`
- 削除したファイル: `path/to/file`

### 実行したコマンド
```bash
# 主要なコマンドを記載
```

### コード変更
主要な変更点をコードブロックで記載。

## 📝 生成・作成したコード/設定

重要なコードや設定ファイルの内容を記載。

## ✅ 解決した課題

- [ ] または [x] 形式でタスクの完了状況を記載

## 🚀 次のステップ・TODO

今後の課題や推奨アクションがあれば記載。

## 🔗 関連リンク・参考資料

参照したドキュメントやURLがあれば記載。

## 💡 学び・メモ

このセッションで得られた知見やTipsがあれば記載。
```

### 4. ファイルの保存
- 生成したMarkdownファイルを `.chat-history/` に保存
- 保存完了後、ファイルパスを表示

## 出力例
```
✅ チャット履歴を保存しました！
📁 .chat-history/20251227-Cursorスラッシュコマンド-チャット履歴保存機能の実装.md
```

## 注意事項
- 機密情報（API キー、パスワードなど）が含まれていないか確認し、必要に応じてマスキング
- 長すぎるログ出力は要約してOK
- コードブロックは適切な言語タグを付ける
